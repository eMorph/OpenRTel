# OpenRTel

A simple mount control program for a DIY H-line radio telescope.

## Building
This is a Python project, thus doesn't require "building" per se, but environment setup does require the build pipeline. I personally recommend using Gnome Builder's default pipeline and then simply launching the result.

##Dependencies
- Python
- GTK4
- libadwaita
- RPi.GPIO/equivalent
- RpiMotorLib
